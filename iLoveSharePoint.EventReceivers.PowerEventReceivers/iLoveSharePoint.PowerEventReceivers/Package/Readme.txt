*iLove SharePoint*
PowerEventReceivers 2.0

Requirements:
-SharePoint Services 3.0 SP1
-PowerShell 1.0
-.NET 3.5

1) Deploy iLoveSharePoint.PowerEventReceivers.wsp either with deploy.bat or with stsadm. 
By default deploy.bat deploys the solution to http://localhost:80
2) Activate the feature on the sites you need it.
3) To edit the scripts you have to be in the farm administrators group
